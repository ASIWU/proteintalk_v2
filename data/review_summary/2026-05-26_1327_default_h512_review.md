## Rationale
- h512 full-suite results were better for the default one-size setting:
  - stronger primary single unseen-drug AUPRC than h768;
  - slightly stronger double unseen-drug pair AUPRC than h768;
  - stronger MSE and graph-feature ablation gaps;
  - faster wall time than h768.
- The validated h512 full-suite run used `LEARNING_RATE=2e-4`, so the default launcher learning rate was aligned to that setting.
